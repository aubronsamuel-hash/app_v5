# Project Helper Scripts

This repository includes a set of PowerShell scripts to automate daily tasks on Windows 10.

## Usage

Open PowerShell in repo root and run:

```
.\shortcuts\<command>.ps1 [args]
```

## Available commands

- `clone.ps1`
- `update.ps1`
- `pr.ps1`
- `tests.ps1`
- `docker.ps1`
- `codex.ps1`
- `tools.ps1`
- `clean.ps1`
